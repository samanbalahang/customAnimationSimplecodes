Dynamic Wave Animation

This design features animated SVG waves with horizontal flipping. Each wave layer has its own animation timing, creating a fluid, organic motion that mimics real ocean waves.

The horizontal flip is achieved using CSS transform: scaleX(-1), and the wave motion is created by animating the horizontal position and vertical scale of the SVG elements.

CSS Animation Code

@keyframes waveAnimation { 0% { transform: translateX(0) scaleY(1); } 50% { transform: translateX(-5%) scaleY(1.1); } 100% { transform: translateX(5%) scaleY(0.9); } } @keyframes flipHorizontal { 0%, 100% { transform: scaleX(1); } 50% { transform: scaleX(-1); } } .wave-1 { animation: waveAnimation 8s ease-in-out infinite alternate, flipHorizontal 16s ease-in-out infinite; } .wave-2 { animation: waveAnimation 10s ease-in-out infinite alternate-reverse, flipHorizontal 14s ease-in-out infinite; } .wave-3 { animation: waveAnimation 12s ease-in-out infinite alternate, flipHorizontal 18s ease-in-out infinite; }

Click the buttons above to control the animation. The waves will flip horizontally and move in a wavy pattern.
